# Want to contribute ?
Please follow these guidelines in order to ensure that your contribution will be accepted.

 - No shady tools must be involved in the contribution
 - No external parsers should be involved in the contributions
 - Nothing outside of STL Should be used
 - No windows API is allowed (Only linux focused toolchains are acceptable)
 - No sudo / root access must be taken unless necessary
 - No intentions of injecting malware should be there with the contributions
 - Any language in comments which can be interprited as racism or harrasments or targetting an indivisual is strictly not allowed.

## What is encouraged but not forced

 - Please use proper comments
 - Please don't use corporate language. Just be normal and casual so others can easily understand the comments
 - Please write comments with your username before and after the part you contributed as it helps in knowing the source of contributions

Thank you.
